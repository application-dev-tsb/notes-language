#Swift Data Types

##Constants and Variables

```swift
//this is a variable
var myVar: Int = 1

//this is a constant
let myConst: Int = 1

//error:
myConst = 2
```

##Type Inference
```swift
//the type declaration is optional
var myInt = 1
```

##Type Conversion
```swift
//int
let twoThousand: UInt16 = 2_000
```

##Basic Types
```swift
Int
Double
Float
Bool

var myInt: Int
```

Type  | Description
------------- | -------------
